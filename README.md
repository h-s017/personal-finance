# 個人財務控管

一個免登入、資料留在瀏覽器內的個人財務 Web App，適合以 GitHub Pages 部署。

## 功能

- 財務總覽：現金存款、信用卡欠款、分期餘額、貸款餘額、淨資產
- 每日收支：收入、消費、信用卡繳款、貸款還款與其他現金流
- 現金存款：帳戶餘額、保留用途、可動用餘額
- 信用卡：未出帳、本期應繳、分期本金、額度使用率
- 分期與貸款：本金、期數、月付、剩餘本金與繳款日
- 月報表：每日支出月曆、年度月報與預算差額
- JSON 匯出／匯入備份

## 記帳原則

1. 信用卡刷卡當日以「消費」認列生活支出。
2. 實際繳卡費時，另記「信用卡繳款」，只影響現金流，不重複計入消費支出。
3. 貸款還款同樣獨立列為現金流出。
4. 信用卡未出帳、本期帳單、分期本金及貸款餘額，依銀行 App 定期更新。

## 部署

此專案為純靜態網頁。將 GitHub Pages 的來源設為 `main` 分支根目錄即可。

> Settings → Pages → Build and deployment → Deploy from a branch → `main` / `(root)`

## 隱私

資料使用 `localStorage` 儲存在目前瀏覽器，不會自動上傳至 GitHub 或其他伺服器。清除瀏覽器資料前，請先匯出 JSON 備份。
